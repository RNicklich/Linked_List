# Linked_List
Implementation of a linked list copy constructor using C++

Homework assignment for ECGR3180 - Data Structures and Algorithms in C++
COmpleted by Rene Nicklich at the University of North Carolina at Charlotte, Fall 2019

In this assignment the students were tasked with implementing a linked list copy constructor in the list.cpp file. 
list_test.cpp and list.h were given, and are compiled together with list.cpp for the correct implementation.
